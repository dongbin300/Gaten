Basic Concept

Candle
캔들차트에서의 하나의 틱을 나타냅니다. 차트의 주기에 따라서 하나의 캔들이 나타내는 시간은 상이합니다.
값설명
candle.open캔들의 시가를 반환합니다.
candle.high캔들의 고가를 반환합니다.
candle.low캔들의 저가를 반환합니다.
candle.close캔들의 종가를 반환합니다.
Volume
캔들차트에서의 하나의 캔들에 포함되는 거래량입니다. 거래가 많이 일어난 캔들일수록 거래량이 높습니다.
값설명
volume캔들의 거래량을 반환합니다.
Interval
캔들차트에서의 하나의 캔들의 시간입니다. 예를 들어 "5m" 일 경우 하나의 캔들은 5분의 시간을 표현합니다.
다음은 사용가능한 주기 리스트입니다.
값설명
1m1분
3m3분
5m5분
15m15분
30m30분
1H1시간
2H2시간
4H4시간
6H6시간
8H8시간
12H12시간
1D1일
3D3일
1W1주
1M1달
Element
값을 나타내는 최소 단위입니다.
Chart Element
차트와 관련된 object나 indicator에 대한 엘리먼트입니다.
값설명
position
candle캔들
volume거래량
riRI 지표
macdMACD 지표
rsiRSI 지표
ma이동평균선 지표
ema지수이동평균선 지표
bb볼린저밴드 지표
Named Element
차트 엘리먼트를 커스텀한 이름을 가진 엘리먼트입니다.
Range Element
값의 범위를 나타내는 엘리먼트입니다.
Trade Element
매매와 관련된 값에 대한 엘리먼트입니다.
값설명
roe등락률
upnl
Value Element
값을 나타내는 엘리먼트입니다.
Scenario
시장은 변동성이 크고 예측불가능하기 때문에 매매 모델에는 상황에 따라 여러 개의 시나리오가 필요합니다. 각각의 시나리오는 독립적입니다. 시나리오 이름은 중복될 수 없습니다.

Basic Info Section

기본 정보를 명시하는 영역입니다.
Asset
초기 자산을 명시합니다.
asset = 25000
다음과 같은 방식으로도 명시할 수 있습니다.
asset = 25k
Period
백테스트 기간을 명시합니다.
다음은 2021년 10월 24일 17시 30분부터 1년 1개월 15일 3시간 5분동안 백테스트 기간을 명시하는 코드입니다.
period = 2021,10,24,17,30:1,1,15,3,5
Interval
캔들의 주기를 명시합니다.
주기에 대한 자세한 정보는 여기입니다.
interval = 5m
Target
백테스트 타겟 심볼을 명시합니다.
현재 타겟은 단일 심볼만 가능합니다.
target = BTCUSDT
